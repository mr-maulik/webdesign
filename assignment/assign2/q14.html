14. What is the use of a span tag? Explain with example?

The <span> tag in HTML is a generic inline container that is used to group and apply styles or manipulate content within a line of text. It does not add any semantic meaning to the content, but it can be styled or manipulated using CSS or JavaScript. The <span> tag is typically used to target a specific part of text or apply styles to a small section of content.

example

<p>This is a <span style="color: blue;">blue</span> text.</p>

This is a highlighted text.
