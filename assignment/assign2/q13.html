13.What is the use of an iframe tag?

The <iframe> tag in HTML is used to embed another HTML document or webpage within the current HTML document. It stands for "inline frame" and acts as a container that can display content from another source within a web page. The content displayed within an <iframe> can be a separate HTML document, an embedded video, a map, or any other web content.

example

<iframe src="https://www.example.com" width="400" height="300" frameborder="0"></iframe>
