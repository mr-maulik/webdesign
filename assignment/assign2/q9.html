9.How is Cell Padding different from Cell Spacing?

-> Cell Padding: Cell padding refers to the space between the content of a table cell and the cell border. It is controlled using the cellpadding attribute in the <table> tag or the padding CSS property. The value of cellpadding can be set to a specific number of pixels or as a percentage of the cell width. For example:

<table cellpadding="5">
    <tr>
       <td>This is a table cell with cell padding</td>
   </tr>
</table>

-> Cell Spacing: Cell spacing refers to the space between adjacent cells in a table. It is controlled using the cellspacing attribute in the <table> tag or the border-spacing CSS property. The value of cellspacing can be set to a specific number of pixels or as a percentage of the table width. For example: html

<table cellspacing="10">
    <tr>
     <td>Cell 1</td>
     <td>Cell 2</td>
   </tr>
</table>
