Q11.What are all the types of Pop up boxes available in JavaScript

JavaScript has three kind of popup boxes: Alert box, Confirm box, and Prompt box.

<script>
alert("Welcome My Javascript Program");
prompt("Welcome My Javascript Program");
confirm("Welcome My Javascript Program");
</script>
