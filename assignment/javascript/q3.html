Q3. What is negative Infinity?

NEGATIVE_INFINITY is "a number lower than any other number".

<script>
    var a = -10;
    var b = 0;
    document.write(a / b);
    document.write(-a / b);
    (output : Infinity)
    (output : Negative Infinity)
</script>
