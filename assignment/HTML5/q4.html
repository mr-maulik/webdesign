q4.Canvas and SVG tags

  • canvas

    • Canvas is a drawing surface provided by HTML5

    It provides a pixel-based rendering approach, where you can draw shapes, images, and animations directly onto the canvas element.

    This browser isn't support canvas!
    EXAMPLE
    <canvas height="150px" width="200px" style=" border: 1px solid black;" id="can">
    </canvas>


    <script>
           var can = document.getElementById('can');
           var ct = can.getContext("2d");
           ct.beginPath();
           ct.rect(10,10,70, 70);
           ct.stroke();
           ct.fillStyle = "lightblue";
           ct.fill();
    </script>

  • SVG

    • SVG stand for scaleble vector graphics

    SVG use pxl

    SVG uses mathematical descriptions of shapes and lines

    EXAMPLE
    <svg width="200" height="100"> <rect x="10" y="10" width="180" height="80" fill="blue" /> </svg>
