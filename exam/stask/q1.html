1.What are HTML Attributs explain with two examle?

HTML attributes provide additional information about HTML elements and are always included in the opening tag.

1.class

The class attribute is used to specify one or more class names for an HTML element. It is often used to apply CSS styles to multiple elements.

< p class="important-text">This is an important paragraph.< /p >

Src Attribute

The src attribute is used with the < img > tag to specify the source (URL) of an image.

< img src="image.jpg" alt="A beautiful landscape" >
